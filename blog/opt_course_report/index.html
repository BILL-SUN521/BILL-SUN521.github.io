    影响力投资——广义上定义为不仅考虑财务目标，还考虑支持某些社会优先事项和议程的其他目标的投资，近年来引起了越来越多的关注。这个概念最初是通过平民主义者努力引入的，通过鼓励机构投资者从从事被批评者视为不道德、不道德或其他令人反感的业务的公司中撤资（例如，剥削童工;默许支持种族隔离或宗教迫害;或赌博、色情、酒精、烟草和枪支业务（统称为“罪恶股票”））。然而，影响力投资的范围已显著扩大，包括使用环境、社会和治理 （ESG） 标准的投资产品、“绿色”债券，以及寻求社会影响力和财务回报的私募股权基金。

    影响力投资的受欢迎程度和管理的资产的增长也引发了强烈反对。例如，2022 年 8 月 4 日，19 个州的总检察长向贝莱德发送了一封由该州的总检察长签署的信函，表达了对资产管理公司的 ESG 政策以及这些政策可能如何影响其持有的化石燃料能源公司的担忧："贝莱德针对各种治理目标的行动可能违反多个州的法律。McCombe 先生的信声称遵守我们的信托法，因为贝莱德的私人动机与其公开承诺和声明不同。这可能不足以满足要求仅关注财务回报的州法律"。鉴于违反受托责任的法律处罚涉及个人弥补客户遭受的任何损失，并将受托人向所述客户提供服务所获得的任何利润退还给客户，因此这些都不是小问题。3

    那么，我们如何协调影响力投资与信托义务呢？答案在于开发一个框架，在这个框架中可以衡量和披露影响力投资的财务影响，这也是本文的主题。

    传统我们通常能想到将影响力投资视为一个标准的投资组合选择问题，其中包含与标的证券的社会影响程度相关的额外限制，因此意味着与不受约束的情况相比，风险/回报状况并不优越。鉴于约束投资组合包含无约束版本的证券的适当子集，数学逻辑表明，约束最优充其量等于无约束的最优，或者更有可能是劣等。

    然而，约束最优的非优越性依赖于一个几乎从未明确说明的关键假设;假设该约束在统计上独立于证券的回报。在某些情况下，这样的假设是有道理的;想象一下，使用包含质数的统一证券识别程序委员会 （CUSIP） 标识符构建一个证券子集。显然，这种约束与任何证券的回报无关;因此，施加这样的限制只会降低优化投资组合的风险调整后回报。

    如果约束不独立于 returns 怎么办？例如，考虑以下约束条件：“仅投资于那些在未来 12 个月内股价将升值超过 10% 的公司。除了施加这种条件的不可行性之外，很明显，这种限制实际上会增加优化投资组合的风险调整后回报。因此，影响力投资的影响是什么这个问题的答案完全取决于影响力标准是否以及如何与所考虑的证券的表现特征相关。

    在本文中，我们开发了一个通用框架来量化影响力投资的财务影响。我们将影响力投资正式定义为根据影响力因子\(X_i\)(对\(X_i\)，因此\(X_i\)越高影响力越大)对 N 只证券的投资可行域进行排序和选择（例如，碳排放量更低、可持续性更高、ESG 评分更高等）。因此，在其他条件相同的情况下，影响力投资者被假定更喜欢\(X_i\)值更高的证券。该影响因子定义了可行域中所有证券的等级排序，从中可以构建影响力投资组合（即 ESG 排名证券的前十分位数或碳排放排名证券的后十分位数）。因此，对投资业绩的影响是由影响指标的向量 X ≡ [X1 X2 ⋯ XN]T 与各个证券的投资业绩的联合分布决定的。

    To formalize this idea, we first propose a general linear multifactor model for asset returns and define excess returns or “alphas” as nonzero intercepts that we model as mean-zero random variables. This framework allows for the possibility of superior investment performance for individual securities but also includes the conventional case of equilibrium or no-arbitrage pricing if we  set the variance of the alphas to zero. In fact, the implications from our model are broadly applicable to an equilibrium asset-pricing setup where “alpha” is reinterpreted as omitted factors of which investors are either unaware or unable to access as easily as professional portfolio managers. Such an agnostic approach to investment performance allows us to determine conditions under which impact investing does and does not change the risk/reward profile of a given investment product.

    为了正式化这个想法，我们首先提出了一个用于资产回报的通用线性多因子模型，并将超额回报或“alphas”定义为非零截距，我们将其建模为均值零随机变量。该框架允许单个证券获得卓越投资业绩的可能性，但如果我们将 alpha 的方差设置为零，则还包括均衡或无套利定价的传统情况。事实上，我们模型的影响广泛适用于均衡资产定价设置，其中“阿尔法”被重新解释为被忽略的因素，投资者要么不知道，要么无法像专业投资组合经理那样容易获得这些因素。这种不可知的投资表现方法使我们能够确定影响力投资在哪些条件下会改变特定投资产品的风险/回报状况。

    特别是，我们在有限样本和渐近（随着证券数量无限增加）中推导出了根据其影响因子 X 排名的单个 alpha 的分布。众所周知，已排序的随机变量（称为顺序统计量）的分布与其未排序的版本具有不同的分布。然而，就我们的目的而言，该文献中更相关的部分侧重于诱导顺序统计，其中随机变量不是根据它们自己的值而是根据其他随机变量的值进行排名（例如，不是根据它们的回报而是根据基金的市场贝塔值对一组共同基金的回报进行排名）。我们使用诱导订单统计数据的属性来推导出影响力投资组合的 alpha 分布，按任意影响力评分 X 排名，使我们能够量化影响力投资的影响。

    使用这个框架，我们表明诱导排序的预期 alpha 由三个项决定：X 与单个证券的 alpha 之间的相关性、单个证券 alpha 的标准差，以及捕捉证券影响因子是高于还是低于平均水平的横截面标准化影响分数。此外，我们还提供了来自诱导排序的预期 alpha 的替代特征，作为基于 alpha 订购证券的预期 alpha 的折扣版本（即，通过一个全知的预言机，实际上，这当然是无法实现的，因为 alpha 是不可观察的）。与 Sharpe-Lintner Capital 资产定价模型 （CAPM）（Sharpe 1964，Lintner 1965）非常相似，该模型通过市场贝塔量化单个证券的预期回报，4 这个简单而深刻的结果突出了影响因子获得超额回报的机制;它在完全知识的情况下实现了最大可能 alpha 的一小部分，其中该分数只是 X 与单个证券的 alpha 之间的相关性。

    利用这一见解，我们量化了基于影响因子 X 形成的投资组合的 alpha——包括从顶部或底部影响因子分位数创建投资组合的常见启发式方法——然后应用 Treynor 和 Black （1973） 框架在形成多头/空头和仅多头投资组合时得出最佳权重，以最大化夏普比率。我们表明，只要影响因子 X 与单个证券的未观察到的 alpha 呈正相关，这种影响力投资组合就与“超高效边界”相关。我们还对我们的结果提供均衡/无套利解释，其中超额回报来自投资者可能不知道但投资组合经理可以访问的影响力因素。在这种情况下，超额回报就投资者观察到的因素而言只是“超额”，并代表了特定影响因素的风险溢价。

    Treynor-Black 投资组合使我们能够构建影响力投资的财务影响的自然衡量标准;当影响因子与单个证券未观察到的 Alpha 呈正相关时，它就具有正 Alpha。另一方面，当影响因子与 alpha 呈负相关并且投资者剥离排名靠后的证券（由于与 X 呈负相关，这些证券平均具有正 alpha）时，影响因子可能会产生成本（在我们的框架中也可以量化）。这为关于采用影响投资效果的不一致且有时相互矛盾的实证发现提供了一个可能的解释。影响因子与 Alpha 之间的相关性受不同影响指标的影响，5 不同的市场条件，6 以及不同的 Alpha 资产定价模型，7 所有这些都会影响影响力投资收益或成本的最终估计。

    为了说明我们结果的实际相关性，我们将我们的框架应用于五个特定的影响力投资背景。每种形式的影响力投资的具体相关性（正或负）取决于影响力的具体性质、实现该影响力所涉及的风险，以及它与阿尔法产生基础过程的关系。

    第一个例子是生物技术风险慈善事业 （VP），这是一种特殊形式的生物医学影响力投资，非营利组织和使命驱动型组织为推进其目标的倡议提供资金，并可能获得可以再投资于其使命的回报。我们以 Kim 和 Lo （2019） 关于囊性纤维化 （CF） 基金会的案例研究为例，表明通过推进罕见病的药物开发可以实现显着的正 α，这说明了“通过做好事来做好事”的可行性。在这种情况下，从财务角度来看，与早期药物开发计划相关的挑战 Fagnan 等人 （2013） 强调的成功概率低、时间跨度长和资本需求大——被正相关 ρ 所抵消。

    第二个申请涉及研发 （R&D） 财团，这是另一种形式的影响力投资。我们以半导体制造技术公司（“Sematech”）为例，这是一个成立于1987年，由美国政府和14家美国半导体公司资助的知名研发联盟，其目的是通过利用共享的研发资源来解决常见的制造问题。使用简单的双重差分方法，我们估计加入该财团会导致成员公司股票的 alpha 增加约 10%，这意味着对该财团的投资将获得可观的回报。在这种情况下，研发联盟减少了研发重复，提高了成员公司的盈利能力（Irwin和Klenow 1996a），从而在影响和回报之间产生了正相关。

    第三个应用涉及衡量从罪恶股票（从事一些人认为在社会上不受欢迎的业务但被记录为具有正 alpha 的公司的股票）中撤资的成本，这可以用 Merton （1987） 被忽视的股票和分割市场的模型来解释。以 Hong 和 Kacperczyk （2009） 以及 Blitz 和 Fabozzi （2017） 为例，我们估计每年放弃的 alpha 的剥离成本在 0.6% 到 3.3% 之间。此示例说明了预估 Alpha 量级对所使用的特定资产定价模型的依赖性，这是所有效果归因练习中众所周知的问题。

    第四，我们将我们的框架应用于几项 ESG 实证研究。这些研究中的具体 ESG 指标与单个证券未观察到的 alpha 之间的相关性决定了对 ESG 投资收益（或成本）的最终估计。在某些市场条件下，它们的范围从债券的 0.05%（Baker 等人，2022 年）到股票的 2.65%（Bansal 等人，2022 年）。这凸显了资产类别、影响力衡量指标和特定市场条件在确定影响力投资的阿尔法方面的重要性，以及需要合理化 ESG 对资产价格的高度动态影响，超越 ESG 投资的均衡模型。

    最后，我们应用我们的框架来解释 2021 年 1 月 GameStop Corp. （GME） 和其他“模因”股票的价格飙升，其中利用机构投资者空头头寸的去中心化空头挤压导致其价格在崩盘前急剧上涨。将此类现象归类为影响力投资似乎很奇怪，但它们在回报如何机械产生方面与其他示例有相似之处，也许，一个名为“价格-影响力投资”的单独类别会更合适。我们的框架有助于量化价格影响投资的财务影响。衡量此类投资的规模并了解其财务影响可以更好地让监管机构和政策制定者了解这种现象的范围和严重性，以便他们能够投入适当的资源来解决它。

    总体而言，这五个例子强调了选择基线资产定价模型并根据经济、制度和市场原理形成影响变量的重要性，以便在影响和回报之间建立健全和稳健的关系。我们的模型提供了一个统一的框架，可以针对各种环境进行校准，并为如何思考影响力投资和量化其财务后果提供了学术严谨性。此外，我们的框架为投资组合经理提供了一种系统且政治中立的方法，以披露影响力投资的财务后果，从而解决其客户不知道或不愿意承担其经理影响力标准后果的任何担忧。
