    Selenium 库和百度

    试着用selenium库写个来模拟在百度中搜索的情况，如搜索“人工智能”并爬取返回的第一页网页的大致信息，如链接网址，标题，日期等内容¶

    下面是一个使用 Selenium 库模拟在百度中搜索“人工智能”的示例代码，并提取返回第一页的搜索结果信息（如链接网址、标题等）。确保你已经安装了 Selenium 和 ChromeDriver。

    安装必要库¶

    首先，确保你已安装 selenium 和 requests（用于后续处理），可以使用以下命令安装： 

    pip install selenium

    示例代码¶

    以下是一个示例 Python 脚本：

    from selenium import webdriver
from selenium.webdriver.common.by import By
import time

driver = webdriver.Chrome()

try:
    # 打开百度
    driver.get("https://www.baidu.com")

    # 查找搜索框并输入搜索内容
    search_box = driver.find_element(By.NAME, "wd")
    search_box.send_keys("冯兴东")

    # 提交搜索
    search_box.submit()

    # 等待搜索结果加载
    time.sleep(2)  # 可以根据网络情况调整等待时间

    # 获取搜索结果
    results = driver.find_elements(By.CSS_SELECTOR, '.result.c-container')

    # 提取信息
    for result in results:
        title = result.find_element(By.TAG_NAME, "h3").text
        link = result.find_element(By.TAG_NAME, "a").get_attribute("href")

        # 有些结果可能没有日期信息，可以选择性提取
        try:
            platform = result.find_element(By.CLASS_NAME, "c-color-gray").text
        except:
            platform = "无平台信息"

        try:
            intro = result.find_element(By.CLASS_NAME, "content-right_2s-H4").text
        except:
            intro = "无简介信息"

        try:
            date = result.find_element(By.CLASS_NAME, "c-color-gray2").text
        except:
            date = "无日期信息"

        print(f"标题: {title}")
        print(f"链接: {link}")
        print(f"简介: {intro}")
        print(f"平台: {platform}")
        print(f"日期: {date}")
        print("-" * 40)

finally:
    # 关闭浏览器
    driver.quit()

    代码说明¶

    1. 导入库：导入所需的库，包括 Selenium 和时间处理。
    2. 设置 ChromeDriver 路径：确保你将 driver_path 替换为你本地的 ChromeDriver 的实际路径。
    3. 打开百度：使用 get 方法打开百度首页。
    4. 输入搜索内容：找到搜索框，输入“人工智能”，并提交搜索。
    5. 等待加载：使用 time.sleep() 等待页面加载，可以根据实际情况调整等待时间。
    6. 提取搜索结果：使用 CSS 选择器获取所有搜索结果，并提取标题、链接和日期。
    7. 输出结果：打印搜索结果的信息。
    8. 关闭浏览器：确保在脚本结束时关闭浏览器。

    注意事项¶

    • ChromeDriver：确保你的 Chrome 浏览器版本和 ChromeDriver 版本匹配。
    • 网络状况：如果页面加载缓慢，可以适当增加 sleep 时间。
    • 异常处理：在提取日期时加入异常处理，以防某些搜索结果没有日期信息。

    你可以根据需要进一步处理提取到的数据。
