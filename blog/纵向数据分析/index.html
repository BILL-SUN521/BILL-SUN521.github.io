    虾类体重数据分析报告¶

    1. 引言¶

    本报告旨在分析虾类体重随时间的变化，并探索其他因素（如种群、性别和水池）的影响。使用了线性混合效应模型（Linear Mixed-Effects Models, LME）进行分析，以考虑重复测量数据中个体差异的随机效应。

    2. 模型设置¶

    2.1 基础模型 (Model 1)¶

    基础模型包括以下固定效应和随机效应：

    • 固定效应：时间对数（Log_Time）、种群（PopID）、性别（SexID）、水池（TankID）。
    • 随机效应：在个体水平上考虑时间对数（Log_Time）的随机截距和随机斜率。

    模型公式为：

    \[ \text{Weight} \sim \text{Log\_Time} + \text{PopID} + \text{SexID} + \text{TankID} + (1 + \text{Log\_Time} | \text{AnimalID}) \]

    2.2 添加交互效应的模型 (Model 2)¶

    为了进一步探索不同因素随时间的影响，Model 2 在基础模型的基础上添加了交互效应，包括：

    • 时间与种群的交互效应（Log_Time:PopID）
    • 时间与性别的交互效应（Log_Time:SexID）
    • 时间与水池的交互效应（Log_Time:TankID）

    模型公式为：

    \[ \text{Weight} \sim \text{Log\_Time} + \text{PopID} + \text{SexID} + \text{TankID} + \text{Log\_Time:PopID} + \text{Log\_Time:SexID} + \text{Log\_Time:TankID} + (1 + \text{Log\_Time} | \text{AnimalID}) \]

    笔者最后使用广义估计方程（GEE）方法对数据集进行分析，研究了不同因素对虾体重（Weight）的影响。我们主要考虑了时间（Log_Time）、种群（PopID）、性别（SexID）、水箱（TankID）及其相互作用项对体重的影响。本次分析分别采用了三种不同的工作相关结构：独立结构（Independence）、可交换结构（Exchangeable）和自回归结构（AR(1)）。

    2. 模型设定¶

    模型公式如下：

    \[ \text{Weight} \sim \text{Log\_Time} + \text{PopID} + \text{SexID} + \text{TankID} + \text{Log\_Time:PopID} + \text{Log\_Time:SexID} + \text{Log\_Time:TankID} \]

    其中：

    • Log_Time：时间的对数值。
    • PopID：种群类别（Pop1, Pop2, Pop3, Pop4）。
    • SexID：性别（Female, Male）。
    • TankID：水箱类别（T1, T2）。
    • 交互项（如Log_Time:PopID）：表示时间和其他因素的联合效应。

    4. 结论¶

    通过不同相关结构的GEE模型拟合，我们发现：

    1. 时间（Log_Time）对虾体重有显著正向影响。
    2. 雄性虾（SexIDMale）的体重大于雌性虾，但增长率较低。
    3. 不同种群（PopID）和水箱（TankID）对体重有显著差异。
